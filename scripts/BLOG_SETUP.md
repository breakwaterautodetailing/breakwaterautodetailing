# Automated Blog Setup

Monthly blog posts are auto-generated by `generate_blog.py` using **OpenAI GPT-4o** and **NewsAPI**.

---

## How It Works

1. On the **1st of every month**, GitHub Actions runs `generate_blog.py`
2. The script fetches recent Broward/Miami-area news headlines from NewsAPI
3. GPT-4o writes a 700–950 word blog post connecting local news to car detailing value
4. A new HTML file is saved to `blog/YYYY-MM-slug.html`
5. `blog/posts.json` is updated with the post metadata
6. `blog.html` listing page is regenerated with the new card at the top
7. Changes are committed and pushed automatically

---

## One-Time Setup (GitHub Secrets)

Go to your GitHub repo → **Settings → Secrets and variables → Actions → New repository secret**

| Secret Name | Where to get it |
|-------------|----------------|
| `OPENAI_API_KEY` | [platform.openai.com/api-keys](https://platform.openai.com/api-keys) |
| `NEWS_API_KEY` | [newsapi.org](https://newsapi.org) — free tier allows 100 req/day |

---

## Run Manually

Trigger anytime from **GitHub Actions → Monthly Blog Post Generator → Run workflow**.

Or locally:

```bash
cd BreakwaterAutoDetailing
pip install -r scripts/requirements.txt
export OPENAI_API_KEY=sk-...
export NEWS_API_KEY=abc123...

# Preview without writing files:
DRY_RUN=1 python scripts/generate_blog.py

# Full run (writes files):
python scripts/generate_blog.py
```

## Post Content Topics (auto-selected by AI)

The AI picks the most relevant angle based on current local news:

- UV damage / South Florida sun and paint protection
- Pollen season, salt air, environmental contaminants
- Hurricane/storm season vehicle prep
- Protecting resale value through regular detailing
- Mobile detailing convenience for South Florida professionals
- Seasonal care (rainy season, snowbird season, summer heat)
- Local events and why your car should look great

## Adding New Posts to Sitemap

After the script runs, manually add the new post URL to `sitemap.xml`:

```xml
<url>
  <loc>https://breakwaterautodetailing.com/blog/YYYY-MM-slug.html</loc>
  <lastmod>YYYY-MM-01</lastmod>
  <changefreq>never</changefreq>
  <priority>0.7</priority>
</url>
```

Or automate this by extending `generate_blog.py` to update `sitemap.xml` as well.

## File Structure

```
blog/
  posts.json                          ← metadata for all posts
  south-florida-spring-sun-...html    ← individual post files
  YYYY-MM-slug.html                   ← auto-generated monthly

scripts/
  generate_blog.py                    ← AI agent script
  requirements.txt                    ← pip dependencies
  BLOG_SETUP.md                       ← this file

.github/workflows/
  monthly-blog.yml                    ← GitHub Actions schedule
```
